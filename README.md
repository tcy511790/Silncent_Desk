# Silncent_Desk
静桌面是一款基于 Tauri + React + Rust 构建的透明桌面叠层应用，定位为“桌面之上的轻量交互层”。其核心在于不打扰日常使用，为用户提供安静、通透且可记忆的辅助体验。应用采用“空闲态”与“交互态”双模式切换，平时极简静默，触发快捷键后方可交互。  在功能上，静桌面主打信息常驻，于屏幕边缘克制展示时间、天气与专注状态；交互层面则通过 Ctrl+Space 等快捷键一键呼出 AI 对话、命令面板与留档。配合点击穿透技术，空闲时完全不干扰底层鼠标操作。视觉上结合毛玻璃风格与专门设计的 16:10 静系列壁纸，预留 UI 安全区，追求静谧高级的长期视觉感受。后端基于 Rust 支撑，提供音频频谱采集、DeepSeek AI 调用、天气获取与文件授权访问，实现系统的流畅联动。
